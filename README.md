# Dot Files

This is my personal collection of dot files for various apps I use on my machine.

The current list comprises of:

- [aerospace](https://github.com/nikitabobko/AeroSpace)
- [ghostty](https://ghostty.org/)
- [leader_key](https://github.com/mikker/LeaderKey)
- [lvim](https://www.lunarvim.org/)
- [nvim](https://neovim.io/)
- [raycast](https://www.raycast.com/)
- [sketchybar](https://github.com/FelixKratz/SketchyBar)
- [starship](https://starship.rs/)
- [vim](https://www.vim.org/)
- [warp](https://www.warp.dev/)
- [yazi](https://yazi-rs.github.io/)
- [zed](https://zed.dev/)
- [zsh](https://ohmyz.sh/)

## How to use

Pull the `dotfiles` repo down to your local machine and create symlinks for the various directories and files.

```shel
ln -s ~/dev/personal/dotfiles/aerospace/ ~/.config/aerospace
ln -s ~/dev/personal/dotfiles/ghostty/ ~/.config/ghostty
ln -s ~/dev/personal/dotfiles/leader_key/ ~/.config/leader_key
ln -s ~/dev/personal/dotfiles/lvim/ ~/.config/lvim
ln -s ~/dev/personal/dotfiles/nvim/ ~/.config/nvim
ln -s ~/dev/personal/dotfiles/raycast/ ~/.config/raycast
ln -s ~/dev/personal/dotfiles/sketchybar/ ~/.config/sketchybar
ln -s ~/dev/personal/dotfiles/starship/starship.toml ~/.config/starship.toml
ln -s ~/dev/personal/dotfiles/vim/vimrc ~/.vimrc
ln -s ~/dev/personal/dotfiles/vim/vim ~/.vim
ln -s ~/dev/personal/dotfiles/vim/ideavimrc ~/.ideavimrc
ln -s ~/dev/personal/dotfiles/vim/vim ~/.config/.nvim
ln -s ~/dev/personal/dotfiles/warp/ ~/.warp
ln -s ~/dev/personal/dotfiles/yazi/ ~/.config/yazi
ln -s ~/dev/personal/dotfiles/zed/ ~/.config/zed
ln -s ~/dev/personal/dotfiles/zsh/zshrc ./.zshrc
```
